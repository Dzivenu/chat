#Chat Room

This chat room aimes to test functions of 
Node.js, Socket.io integrated with php

We assume you have already node.js installed.

- configuration

$npm install

- launch app

$node app.js


- php nodejs 

$php test.php

- open your browser and go: localhost/chat/ without putting #port
